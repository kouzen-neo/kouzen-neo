### About

Building production-grade applications from scratch -- database design, REST API, and modern frontend. Specialized in the **React** ecosystem (React Native, Next.js, React) with **offline-first** architecture.

```
React Native  ·  Expo  ·  Next.js  ·  Bun  ·  SQLite
TypeScript    ·  Node.js  ·  Docker  ·  Tailwind CSS
```

Primary focus: POS systems, mobile commerce, interactive web applications.

### Projects

**Kouzen POS -- Web**
POS system for small businesses. Zero database configuration, instant startup. Proprietary.

```
Stack   Bun  ·  ElysiaJS  ·  SQLite  ·  React 19  ·  Vite  ·  Ant Design 6
```
